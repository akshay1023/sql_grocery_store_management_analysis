# Grocery Store SQL Analysis Project

## Project Overview
This project analyzes a grocery store database using SQL.  
The analysis focuses on customer behavior, product performance, sales trends, supplier contribution, and employee performance.

## Tools Used
- SQL
- MySQL
- Relational Database Concepts

## Database Tables
The project uses the following tables:

- Customers
- Orders
- Order Details
- Products
- Categories
- Suppliers
- Employees

## Business Questions Solved

### Customer Insights
- Number of unique customers
- Customers with highest orders
- Total and average purchase value per customer
- Top 5 customers by purchase amount

### Product Performance
- Products per category
- Average product price by category
- Highest selling products
- Total revenue generated by products

### Sales and Order Trends
- Total orders placed
- Average order value
- Order trends by date and month
- Weekday vs weekend order patterns

### Supplier Contribution
- Total suppliers
- Supplier providing most products
- Average product price per supplier
- Supplier revenue contribution

### Employee Performance
- Employees processing orders
- Employees handling most orders
- Total sales handled by employees
- Average order value per employee

### Order Details Analysis
- Relationship between quantity and total price
- Average quantity ordered per product
- Unit price variation across products

## File Included
SQL_QUERIES.sql – Contains all SQL queries used for analysis.

## How to Run
1. Import the database into MySQL
2. Run the SQL queries in MySQL Workbench
3. Analyze the results for business insights
